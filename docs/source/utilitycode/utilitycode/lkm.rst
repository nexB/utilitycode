Usage
=====

.. code-block::

    Usage: lkm [OPTIONS] INPUT OUTPUT

    Extract the LKM statements. i.e. linux/module.h, MODULE_LICENSE

    Options:
    -h, --help  Show this message and exit

Example
=======

.. code-block::

   lkm ~/project/ ~/project/scan/lkm.xlsx
